# Transolver with PipeStreamFunctionBoundaryAnsatz on the pipe flow benchmark
# across four data budgets, additionally training on u_y with weight 0.1.
# Mirrors transolver_stream_ansatz_data.txt with `data.load_uy=true` and
# `constraint.uy_loss_weight=0.1` overrides.
#
# Run via the HPC runner:
#   RUNS_FILE=experiments/pipe/transolver_stream_ansatz_uy_data.txt bash scripts/hpc/runner.sh
# Or locally via the batch runner:
#   bash scripts/run_batch.sh experiments/pipe/transolver_stream_ansatz_uy_data.txt
#
# Output layout:
#   outputs/pipe/pipe_stream_function_boundary_ansatz/transolver/<budget>/seed_42/{resolved_config.yaml,best.pt}
